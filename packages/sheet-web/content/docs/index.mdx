# Tiara Stack documentation

TiaraDocs is the reference for servers utilizing Tiara Stack such as TiaraBot for Project Sekai tiering. Start with the task in front of you. This may be checking in, reading your schedule, posting a schedule, replacing a fill, or recovering from a bot problem.
