<Callout type="warn" title="Theerie-controlled feature gate">
  Team submission is unstable. Theerie enables or disables it per server. If it is unavailable, use
  the server’s manual team process and omit every confirmation step below.
</Callout>

Where enabled, TiaraBot watches new and edited messages in the configured team-submission channel. It reacts, parses the filler’s message, writes recognized teams to the sheet, and reports added or skipped teams.

## Confirmation-enabled servers

The write happens **before** the controls appear.

- **Confirm** accepts the written submission and removes the confirmation reply.
- **Reject** attempts to roll back the write.
- If rollback fails, follow the updated reply and inspect/correct the sheet manually.

## Confirmation-disabled servers

TiaraBot writes automatically and returns a status reply without Confirm/Reject controls.

Use `/team list user:<filler>` for a read-only verification of saved tags, lead/backline ISV, optional talent, and effect percentage. It is not an input or edit command.
