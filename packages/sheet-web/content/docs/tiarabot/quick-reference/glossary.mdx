---
title: Tiering shorthand used here
description: A compact map of familiar terms used in TiaraBot instructions.
---

These docs assume you already know tiering and use the following terms without teaching tiering strategy:

- **runner / tierer:** The person grinding event points.
- **filler / support / fill team:** The people supporting the runner’s co-op room.
- **private room / co-op room:** The coordinated room. **Pubbing** is public matchmaking.
- **Fill / Overfill / Standby:** Schedule states returned by TiaraBot.
- **EB:** Event Bonus.
- **ISV / skill value:** Often written lead/total, such as `100/500`.
- **Encore:** The sixth skill proc.
- **Envy / LnF / Melt / HCM:** Familiar tiering-song shorthand. Envy refers to Hitorinbo Envy Expert and LnF to Lost One no Goukoku Hard in this context.
- **Marathon / World Link:** Different event formats that use the same TiaraBot schedule, check-in, and room-order flow.
- **CC:** Cheerful Carnival. Mana/moni must review room order more closely so its combined total does not cross the server’s threshold for avoiding slower matchmaking.
- **battery / energy / crystals:** Runner resources TiaraBot does not manage.
- **T10, T50, T100, T200, T500, T1000, T5k, T10k:** Tier targets, not values reported by `/status` or `/screenshot`.

Use the [How to Tier guide](https://docs.google.com/document/d/1jA03i1D7cTB0rwhaRHHErlDbPoLf1-gP/edit#heading=h.c6q6k29opnkr) for strategy and the [Managing/Monitoring guide](https://docs.google.com/document/d/1fUGQq89a8TMnJQYxiE44TCJu4-FDhK-Tdgr6qPx_vr4/edit?tab=t.0) for management background.
