---
title: Quick command reference
description: Exact commands, role, output, and the state each command does or does not change.
---

For caller access and every slash-command parameter, see [Command permissions and parameters](/docs/tiarabot/command-reference).

| Surface                   | Role                           | Result                                                                                                                                                           | Does not do                                                               |
| ------------------------- | ------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------- |
| `/schedule list`          | Shared                         | One person’s Fill/Overfill/Standby ranges for a day.                                                                                                             | Full schedule image, room code, schedule edit.                            |
| `/screenshot`             | Mana/moni                      | `screenshot.png` of the configured schedule-sheet range for channel/day.                                                                                         | EP/cutoff or in-game screenshot.                                          |
| Check-in button           | Filler                         | Records check-in, restores a configured lockdown role, and updates/announces state.                                                                              | Validate in-game readiness or team.                                       |
| `/checkin manual`         | Mana/moni                      | Creates TiaraBot’s check-in flow at a server-chosen time.                                                                                                        | Serve as a filler fallback or choose emergency policy.                    |
| `/slot list`              | Shared                         | Open Slots and Filled Slots for a day.                                                                                                                           | Claim, assign, or ping a slot.                                            |
| `/slot button`            | Mana/moni                      | Persistent button for refreshed open-slot results.                                                                                                               | Write a schedule-sign-up post.                                            |
| `/room_order manual`      | Mana/moni                      | Sheet-backed interactive room-order draft.                                                                                                                       | Let fillers claim positions.                                              |
| `/kick manual`            | Manage Server or Administrator | Ad-hoc fallback that removes the lockdown role from members not scheduled for the hour; on automatic-check-in servers, automatic cleanup runs separately at :15. | Edit the schedule or replace a filler.                                    |
| `/team list`              | Shared                         | Read-only saved team tags/ISV/talent/effect lookup.                                                                                                              | Submit or edit a team.                                                    |
| Team-submission channel   | Shared, gated                  | Where enabled, parses messages and writes teams.                                                                                                                 | Exist in every server. Stable behavior is not guaranteed.                 |
| `/preference checkin-dm`  | Filler                         | Check-in DM status/enable/disable.                                                                                                                               | Availability or team preference.                                          |
| `/preference monitor-dm`  | Mana/moni                      | Monitor DM status/enable/disable.                                                                                                                                | Change check-in timing.                                                   |
| `/server`                 | Mana/moni                      | Linked sheet, auto-check-in, and monitor-role config.                                                                                                            | General event configuration.                                              |
| `/channel set`            | Manage Server or Administrator | Running flag, logical name, optional lockdown role, check-in destination.                                                                                        | Apply the lockdown permission overwrites or configure every announcement. |
| `/channel lockdown_setup` | Mana/moni or Manage Server     | Replaces a selected channel's overwrites with the standard lockdown-role, monitor-role, and hidden-for-everyone permissions.                                     | Preserve additional role or member-specific channel overwrites.           |
| `/channel lockdown_undo`  | Mana/moni or Manage Server     | Removes all permission overwrites from a selected channel so inherited permissions apply again.                                                                  | Remove the saved Lockdown role from TiaraBot's channel configuration.     |
| `/status`                 | Shared                         | TiaraBot service readiness.                                                                                                                                      | Runner, room, event, sheet, or cutoff status.                             |
