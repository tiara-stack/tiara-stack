Use this section for the TiaraBot part of a fill shift. It assumes you already know the runner’s room plan, your fill-team/ISV requirements, and the expected server procedures.

## Typical path

1. Check your Fill, Overfill, and Standby hours with [`/schedule list`](/docs/tiarabot/fillers/read-your-schedule).
2. Where the submission channel and server feature flag are both enabled, submit the requested fill team and review TiaraBot's reply.
3. Respond to TiaraBot’s check-in message or DM with the **Check in** button.
4. Enter the running channel for the host’s pinned room order and code. TiaraBot restores its access role when the server uses one.
5. If anything is wrong, contact your server’s mana/moni immediately.

<Callout title="TiaraBot does not replace the shift team">
  There is no filler command to claim an open slot, drop yourself from the sheet, or file a generic
  bot issue. Keep mana/moni aware of changes that affect the room.
</Callout>
