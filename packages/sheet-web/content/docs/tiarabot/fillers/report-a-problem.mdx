---
title: Report a TiaraBot problem
description: Give your server’s mana/moni enough context to protect the current room.
---

Tell **your server’s mana/moni** whenever the schedule, check-in, running-room role, team result, or room context is wrong. They need to know the problem exists even when you can work around it.

Include:

- the command, button, or message involved.
- event server and running room.
- day and hour.
- the visible response or error.
- a Discord message link when available.
- whether the issue blocks the current room.

Do not use `/kickout` to remove yourself from the schedule. It is a mana/moni room-role cleanup command, not a slot-drop command.
